# MiniHotel FE

### Installation

1. **Install dependencies**
   \`\`\`bash
   npm install --legacy-peer-deps
   \`\`\`

2. **Start development server**
   \`\`\`bash
   npm run dev
   \`\`\`

3. **Open in browser**
   Navigate to `http://localhost:3000`

### Building for Production

\`\`\`bash
npm run build
npm start
\`\`\`
